# Aplikacje do nagrywania scieżki dzwiękowej

Aplikacja służy do nagrywania scieżki dzwiękowej, przechowywania jej oraz odtwarzania.

Wykorzystane technologie:
* Django
* Videojs-record
* Ajax
* HTML
* Bulma CSS
